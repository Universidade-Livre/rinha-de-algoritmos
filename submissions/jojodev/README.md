## Como Executar

**Pré-requisitos:**
   - Tenha Python instalado.

**Execução:**
   - Abra um terminal no diretório do script.
   - Execute o script com o comando:
     ```
     python main.py input.txt
     ```
   - Exemplo:
     ```
     python main.py ../../resources/instances/problem_01/exemplo_02.txt
     ```
