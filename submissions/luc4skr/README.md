# Como executar

## Pré requisitos

- Python 3.11

## Execução

- No diretório onde esta o código execute o seguinte comando:

### Linux

```
python3 problema01.py args.txt
```

### Windows

```
python problema01.py args.txt
```

### Exemplo

```
python problema01.py ../../resources/instances/problem_01/exemplo_01.txt
```
